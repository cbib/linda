What Linda does ?

Using Linda you can describe your entire experimental design, whether to specify which biological species will be studied during your investigation, what types of technical equipment you will use, what types of environmental conditions will apply or be measured, what experimental factors will be investigated and much more. Linda uses the MIAPPE model to structure the information. The figure below represents the MIAPPE diagram with each of the elements composing it.

Tree representation

Each investigation and each ontology is displayed using tree representation. Each node of the tree is clickable and contains details about selected node or leaf in the tree.

How Linda Does ?

Afin de structurer l'information Linda s'inspire des standards dans le domaine MIAPPE. L'échelon le plus haut de la hiérarchie Linda est l'élément Investigation. Chaque investigation est identifiée par un identifiant unique.

At the very beginning....

Lorsque vous vous connectez pour la première fois sur Linda, aucune investigation n'apparait dans l'arbre des investigations (lien vers projects tab)

Create your first investigation

Pour créer une investigation, il suffit de cliquer sur le associé et un menu apparaitra dans lequel vous pourrez choisir entre trois propostions

Trois choix s'offrent à l'utilisateur

  • Add investigation from original model template

    Le formulaire modèle MIAPPE pour une investigation . La description des champs pour chaque structure du modèle MIAPPE peut être trouvé ici

  • Add investigation from saved template

    L'utilisateur peut sauvegarder un modèle pré-rempli en cochant l'option save template au moment de valider le formulaire. Ces modèles préremplis peuvent ensuite être réutilisés autant dde fois que besoin. Il est cependnt rare de ccréer deux fois la même investigation mais cela peut s'avérer utile pour rejouer des études types/ modèles dans dautres investigations

  • Add investigation from zip files

    L'utilisateur peut sauvegarder l'ensembble d'une investigation avec tous les éléments la composant sous forme de dossieer compressé au fromat zip. Par conséquent l'utilisateur peut également charger l'ensemble d'une investigation sous forme de fichiers .zip.
