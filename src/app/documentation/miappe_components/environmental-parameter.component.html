Linda uses the MIAPPE model to structure the information. The figure below represents the MIAPPE diagram with each of the elements composing it. Each component of the MIAPPE model contains several fields helping to describe your whole investigation

Environment Parameter

Environmental parameters that were kept constant throughout the study and did not change between observation units or assays. Environment characteristics that vary over time, i.e. environmental variables, should be recorded as Observed Variables (0-1 per study).
