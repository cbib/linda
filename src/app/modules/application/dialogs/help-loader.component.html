Steps to follow

1 - Select a file on your local system using the browse button and select the right field delimitor in the list.

2 - Once you have clicked on OK button, your dataset has been loaded and only columns headers are shown.

3 - Assign a column as study column in order to create the backbone of your investigation by select 'Extract Study label from column' to assign this column to the study label. (you can visualize the data related to one column header by clicking on display button)

4 - There are three options to import data.

  • - You could import only study ("Only Studies") and build the simple study components corresponding to the unique study labels found in the seleted column
  • - You have checked the import data option ("Studies + associated data"), consequently all data related to each study will be upload and attached to their parent studies as data files
  • - If your studies are already imported from previous extracction step, you may want to import only related data or metadata. your data files need to contain a column with study labels corresponding to the studies labels in your investigation. To do that, select option "Only Data", it will attached each studies's related data.

Once MIAPPE components assignment is done, check boxes for each component other than study component (observed variables, experimental factors, material sources can be imported this way), you want to be incorporated in your investigation and click on upload button For each study created, each line of your data that matches Study Label will be add as a MIAPPE study component data file extracted and linked to the corresponding study in the investigation tree
