How to create an investigation ?

How Linda Does ?

In order to structure the information, Linda is inspired by standards in the MIAPPE field. The top rung of Linda's hierarchy is the preliminary investigation element. Each survey is identified by a unique identifier. Depending on your experimental design, one or more studies could be associated with. your investigation Each MIAPPE ccomponent can further be added to describe the whole experimental design.

At the very beginning....

Create your first investigation

To create an investigation, simply click on the associated et un menu apparaitra dans lequel vous pourrez choisir entre trois propositions

Three possibilities are available to the user:

  • 1 - Add investigation from original model template

    This option uses the MIAPPE model form for an investigation. The description of the fields for each structure of the MIAPPE model can be found here Only two fields must be completed, the unique identifier associated with the investigation as well as a Project Name.

    Once the mandatory fields have been completed, the investigation can be saved by clicking on the submit button. A new branch will now appear in the investigation tree

    Attention, les identifiants doivent être uniques et un message d'erreur apparaitra si l'identifiant choisi est déjà présent dans la base de données.

    The form can be opened again later in edit mode to complete the missing fields.


  • 2 - Add investigation from saved template

    The user can save a pre-filled template by checking the save template option when validating the form. These pre-filled templates can then be reused as many times as needed. However, it is rare to create the same investigation twice but it can be useful for replaying typical/model studies in other investigations


  • 3 - Add investigation from zip files

    The user can save the whole of an investigation with all the elements composing it in the form of a file compressed in zip fromat. Therefore, the user can also upload an entire investigation as .zip files.



How to load environmental metadata ?
  1. Right click on study node and select add metadata_files. You will be redirect to download page
  2. Select file (csv, excel, tsv, ...) using browse button.
  3. Your file will be displayed in a table
  4. You can drag and drop select list to corresponding headers (mai ly for the Time column)
  5. You can assign an ontology term to each header except time column.

How to remove node in History tree ?
  1. Right click on node and select remove if you want to remove a single node.

  2. If you want to suppress multiple nodes, you need to click on window button setting and activate the multi selection mode. Then you can checked nodes you want to remove and click again on the window settoings button to remove selected
