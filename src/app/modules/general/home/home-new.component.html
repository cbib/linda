Features

Preserve and store environmental metadata

  • Insert metadata respecting the MIAPPE model (https://www.miappe.org/)
  • Use the associated ontologies (Plantontology, XEO, ENVo, etc.)
  • Data from sensors files can be import as tabular and json files

Integrate experimental downstream data from external database (PLATO)

  • Import whole PLATO analysis (PLATO is designed to capture and store experimental data from metabolites, enzymes profiling)
  • Build aggregated tables
  • Extract knowleddge using statisticals tools
  • And many more …

Preserve and store environmental metadata

  • Export projects in various formats: csv, tsv, excel, json, isa-tab
  • Correlate your metadata measurement with experimental design and growth protocol
  • Assign ontology term to your measurment
  • Describes observation unit and climatic events using ontology

Save your work in linda Database

  • Export import whole investigation in json, tsv, zip format
  • Export investigation in iSA tab format
  • Share investigation with other users
  • Save Experimental design informations and environmental metadata in Linda DB
  • Facilitate a standardized exchange of information

Design your experiment in a simple way

  • Duration, start and end date
  • Biological Materials and material sources
  • Experimental factors and observed variables
  • Describes observation units and climatic events
  • And many more …

Use collections on ontologies ready to used

  • CROP ontology
  • Plant ontology
  • XEO ontology
  • ENVo ontology
  • PECO ontology
  • and various other ontologies
