1. How to create an investigation ?
  1. Right click on History node and select add investigation.
  2. Choose between creating Investigation from scratch or using a template if exists.
  3. You will be redirect to the investigation form.

1. How to load environmental metadata ?
  1. Right click on study node and select add metadata_files. You will be redirect to download page
  2. Select file (csv, excel, tsv, ...) using browse button.
  3. Yopur file will be displayed in a table
  4. You can drag and drop select list to corresponding headers (mai ly for the Time column)
  5. You can assign an ontology term to each header except time column.
